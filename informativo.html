Flexbox vs CSS Grid

Escolher layout

Resumo de comportamentos

Unidimenional

O layout é calculado itema à item. Otimo para distribuir espaço em uma linha

O poder do Flexbox

  • Foco no conteudo
  • Ideal para componentes pequenos (menus, botoes)
  • Facilidade em alinhas itens em um unico eixo
  • Facilidade: Itens podem crescer e encolher dinamicamente

O poder do Grid

  • Foco no layoutcontainer
  • Estruturas de paginas completas
  • Controle toal de linhas e colunas simultaneamente
  • Perfeito para galerias aonde o aliamento vertical importa
